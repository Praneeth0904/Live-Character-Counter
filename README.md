## 📌 Live Character Counter  

Displays real-time character count as the user types.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Tracks the number of characters as the user types.
* Updates instantly with each keystroke.
* Useful for input validation and limits.
